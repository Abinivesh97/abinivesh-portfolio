Key takeaways

Four simple truths that make the internet “click”

Remember these

Rules are required for communication.

Models (OSI / TCP‑IP) help us agree on what each part does.

Data gets “wrapped” (encapsulation).

Each layer adds a label to deliver correctly, unwrapped at destination.

Switches = local; Routers = world.

Switch uses MAC (local). Router uses IP (between networks).

Wireshark lets you watch the labels.

Eth → IP → TCP/UDP → TLS show up as readable sections.
