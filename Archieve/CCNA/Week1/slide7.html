Layer 3: Network — The GPS & Post Office

Adds IP addresses and chooses the next “road” toward the destination

OSI Layer 3 of 7

What Layer 3 does

“Where is it going?” and “Which way should it travel?”

Adds logical addresses: IP addresses

Like writing a full street address on a package so it can travel between cities.

Turns segments into packets

A packet is a chunk of data with an IP “shipping label.”

Chooses the next hop (next router)

Routers compare options and forward the packet one step closer.

IP addresses

Like “mailing addresses” for devices

IPv4 (older)

192.168.1.50

Often used inside homes.

IPv6 (newer)

2001:db8::1

More addresses for the growing internet.

Beginner tip

A domain name (example.com) is easier for humans. DNS converts it to an IP address so routers can route it.
