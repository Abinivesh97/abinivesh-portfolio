Encapsulation: Wrapping the Package

Each layer adds a “wrapper” (header) so the data can travel safely

Sender wraps • Receiver unwraps

Plain-English idea

Like packing a fragile item into multiple boxes

Each layer adds a header (label).

Headers contain helpful info like “who is this for?”

Different labels help at different moments.

Switches read MAC (local). Routers read IP (global).

At destination: decapsulation.

Receiver peels off wrappers to reach original data.

Beginner tip

Your message (payload) stays safely inside. Most layers add “outside labels” so it can move.
